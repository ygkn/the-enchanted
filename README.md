# The Enchanted

</div>

> Magic as a Service.

## Develop

### Requirements

- [Node.js](https://nodejs.org/)
  - v14.\*
- [yarn](https://yarnpkg.com/)
  - 1.\*

### Preparation

Clone this repository.

Then, install dependencies with yarn on **repository root**.

```sh
yarn
```

### Run

In each workspaces ( packages/\* ), run `yarn dev`.
